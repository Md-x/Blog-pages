    5 ways to check a websites’ development platform

    Arthur April 7, 2021 Websites

    If you own a website or you are about to own one, you probably want to know what your competitor built their website with or you saw a really nice website with a theme you would like to copy. Well, if you are curious enough to have insights and ideas on what technology was used, here are a few ways you can use to determine what platform a website is built on or with. This is useful for both Chrome and Firefox browser users.

    1. Use a Website Tool

    Several websites exist to make this easy, all you have to do is type in the web address and they will analyze the technology stack. You can add some as an extension to your browser to turn on and off whenever you like. Some of these tools include;

    • Builtwith
    • W3Techs
    • Wappalyzer; Chrome & Firefox extension
    • WhatRuns; Chrome & Firefox extension
    • SimilarTech; Chrome & Firefox extension
    • WTFramework

    The website’s source code can be accessed from your browser. Using chrome, find Developer Tools, and using Firefox, find Web Developer in your menu. The source code's file extensions and URLs can tell you what type of platform the website is built on.

    Some websites give away more information such as the email address behind a website, its operating system, web server, web technologies that are being used. 

    For Chrome Users, here are the methods you could use to access Developer tools. If you are willing to explore the options you have with Chrome, you can learn more here. https://developer.chrome.com/docs/devtools/

    • Visit a website of your choice, then right click on the page, and select ‘inspect’ from the menu.
    • Go to section at the top right in the menu bar, then select ‘more tools’, then to Developer Tools.

    For Firefox users, use these methods.

    • Right click on the website page and select ‘inspect element’ then explore the options for Developer tools.
    • Click on the top right section of your screen to access ‘Developer tools’, then select ‘Web Developer’

    We hope this article has helped you figure out how to satisfy your curiosity and provides insight on how others have achieved their goals with the internet with readily available tools. If you are an IT professional that designs, builds and maintains websites regularly you can easily check a websites’ development platform now.
